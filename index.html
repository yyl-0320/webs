我的收藏网站

Reddit

Reddit 是一个社交新闻网站，用户可以在不同的主题下发布内容并进行讨论，涵盖了几乎所有领域的信息。

Medium

Medium 是一个内容分享平台，用户可以阅读和分享文章，内容涵盖了各种主题，从技术到文学，从生活方式到商业。

Behance

Behance 是一个在线的创意作品分享平台，展示了全球各种领域的创意作品，包括插画、平面设计、摄影等。

Khan Academy

Khan Academy 是一个在线教育平台，提供免费的教育资源，涵盖了数学、科学、历史、编程等各种学科。

Coursera

Coursera 是一个提供在线课程的平台，合作了世界各地的大学和机构，提供了各种免费和付费课程。

GitHub

GitHub 是一个面向开发者的代码托管平台，用户可以在上面存储、共享和管理代码，也是开源项目的主要平台之一。
